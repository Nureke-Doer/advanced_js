# Паттерн: Module (GoF Style)
* **Название:** Module
* **Описание:** Группирует связанные методы и переменные, скрывая детали реализации.
* **Проблема:** Засорение глобального пространства имен и отсутствие приватности.
* **Решение:** Использование замыканий для создания приватных областей.
* **Дизайн:** Публичные методы доступны через возвращаемый объект, приватные — скрыты внутри.
* **Известное использование:** Модульные системы в Node.js и классические JS библиотеки.
