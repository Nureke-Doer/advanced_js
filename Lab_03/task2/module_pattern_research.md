# Исследование паттерна Module
* **История:** Появился как способ инкапсуляции данных до введения официальных модулей в JS.
* **Связь с другими паттернами:** Является основой для Revealing Module Pattern.
* **Использование:** Применяется в библиотеках типа jQuery для создания закрытой области видимости.
* **Современность:** Сейчас чаще используются ES6 модули (import/export).
