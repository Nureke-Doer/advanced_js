# Документация паттерна: CurrencyFormatter
* **Название:** CurrencyFormatter
* **Описание:** Обеспечивает единообразное форматирование чисел в денежный формат.
* **Контекст:** Системы, где важна точность отображения валют для пользователя.
* **Проблема:** Разные части кода форматируют цены по-разному, что ведет к ошибкам интерфейса.
* **Решение:** Создание единой утилиты с заданными параметрами локали.
* **Дизайн:** Функция принимает число и возвращает строку.
* **Реализация:** Применяется стандарт ES6+ Intl.NumberFormat.
* **Последствия:** Улучшает читаемость кода, но требует поддержки современных браузеров.
