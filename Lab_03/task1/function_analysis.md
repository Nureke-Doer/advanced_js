# Анализ функции форматирования
* **Проблема:** Необходимость отображать числа как валюту с фиксированным знаком.
* **Контекст:** Веб-приложения, интернет-магазины, финансовые отчеты.
* **Решение:** Использование встроенного объекта Intl.NumberFormat.
* **Компоненты:** Входное число, код валюты, локаль.
